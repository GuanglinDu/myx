# Coding Standards

All Python code in this project must adhere strictly to the PEP-8 style guide. Key rules include:

- Use 4 spaces for indentation.
- Limit code lines to a maximum of 79 characters.
- Limit comment lines to a maximum of 72 characters.
- Use lowercase with underscores for function and variable names.
- Use two blank lines between top-level functions and classes.
